# EstiMate (by AI-nt No Force)

AI-powered project estimation. Upload an RFP or answer a questionnaire to get detailed timeline, costs, team, and risk assessment. 


## Getting Started

Setup & Run Your Next.js Project

Install dependencies
In your project root, run:

npm install next react react-dom

In the root folder of your project, create a file named .env.local and add your API credentials:

ANTHROPIC_API_KEY=your_api_key_here
ANTHROPIC_BASE_URL=your_base_url_here

Start the development server
Run:

npm run dev

Your app should now be running at http://localhost:3000
